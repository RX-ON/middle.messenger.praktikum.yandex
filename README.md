## Команды для запуска
Для запуска в режиме разработки:
```
$ npm run dev
```
Чтобы выкатить билд в папку dist:
```
$ npm run build
```
Для запуска сервера:
```
$ npm run server
```
Для запуска проекта:
```
$ npm run start
```
